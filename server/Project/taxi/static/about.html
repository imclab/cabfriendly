What is CabFriendly?

CabFriendly is a mobile web application that combines social networking and ride sharing to help you save money and the environment while meeting new people. CabFriendly will match you up with others nearby who are taking similar trips so that you can "Share the Fare". As the site grows, we will be adding many new features to help make ride sharing as easy as possible no matter where you are or where you want to go!

Where did it come from?

CabFriendly was developed by Sergey Karayev, Harold Pimentel, and Adam Roberts as a project for CS262: Advanced Topics in Computer Systems, a graduate computer science course taught by Eric Brewer at UC Berkeley.
